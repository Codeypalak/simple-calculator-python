# Simple Calculator with History

A command-line calculator built in Python that performs basic arithmetic operations and saves calculation history to a file.

## Features
- **Basic Operations**: Addition, Subtraction, Multiplication, Division
- **History Tracking**: All calculations saved automatically in `history.txt`
- **View History**: Type `history` to see past calculations
- **Clear History**: Type `clear` to delete all saved history
- **Exit**: Type `exit` to quit the program

## Installation & Usage
1. Clone the repository
2. Run `python calculatewithhistory.py` in terminal
3. Enter calculations in format: `num1 operator num2` e.g. `8 - 4`
4. Use commands: `history`, `clear`, `exit`
